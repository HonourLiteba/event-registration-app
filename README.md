# Event Registration App

## Project Description
A client-side single-page web application for managing tech event registrations using HTML5, CSS3, and vanilla JavaScript. The app validates user inputs in real time, provides visual success/error feedback, manages a dynamic participant list, and features data persistence along with record deletion capabilities.

## Features Met
- **Task 1 & 3**: Version Control tracking workspace setup and dynamic array data collection storage.
- **Task 2**: Strict conditional check parameters for form inputs with visual UI notifications.
- **Option A**: Dynamic item deletion via structural identification filter arrays.
- **Option D**: Data persistence built directly via browser-backed `localStorage` implementations.

## Project Structure
- `index.html` - Semantic HTML structure for the registration form and participant list display.
- `style.css` - Custom styling utilizing a dark theme with orange highlights, clean input fields, and custom success/error message states.
- `script.js` - Form validation logic, array manipulation, DOM rendering, and localStorage synchronization.
